# Redragon K552P-KBS English Software Translation

This repository contains the fully English-translated configuration and localization files for the **Redragon K552P-KBS** (as well as compatible K628-RGB / K816 OEM series) mechanical keyboard driver. 

The original factory software often leaves several critical strings, macro settings, mouse button assignments, and firmware update prompts untranslated in Chinese or Portuguese. This project fixes that by providing a clean, fully localized English XML file.

---

## 🚀 Features Fixed & Translated

* **Firmware & Error Prompts:** Clear English warnings for device disconnections, firmware updates, and multi-device conflicts.
* **Macro Editor:** Full translation of macro loops, delays, key presses, releases, and batch import/export options.
* **Mouse/Gesture Bindings:** Clear labels for mouse clicks, scrolls, and advanced button functions (e.g., Fire key, Triple click).
* **RGB & Music Modes:** Properly localized names for all 22+ lighting effects and audio-reactive profiles (e.g., "Neon Stream", "Glowing Waves", "Audio Dance").

---

## 🛠️ How to Install

1. Download and extract (unzip) the file.
2. Run the software.
